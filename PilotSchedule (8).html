☁️ Cloud Sync Setup
Connect Supabase for real-time sync across all devices
✕
📋 How to set up (free — takes 3 minutes):
1. Go to supabase.com → Sign up free → New Project
2. Wait ~1 min for project to start → Go to Settings → API
3. Copy your Project URL and anon/public API Key → paste below
4. Go to SQL Editor → run this one command:
CREATE TABLE IF NOT EXISTS app_data ( key TEXT PRIMARY KEY, value TEXT, updated_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE app_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "Allow all" ON app_data FOR ALL USING (true) WITH CHECK (true); ALTER PUBLICATION supabase_realtime ADD TABLE app_data;
5. Paste your credentials below and click Connect ✓
